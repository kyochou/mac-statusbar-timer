# StatusBarTimer

一个简单的 macOS 状态栏计时器应用，用于跟踪工作时间。

## 功能特点

- 在状态栏显示工作时间（小时:分钟）
- 超过 1 小时时间显示为红色
- 锁屏时自动重置计时器
- 轻量级，不占用 Dock 空间
- 右键菜单支持快速退出

## 系统要求

- macOS 12.0 或更高版本
- Xcode 命令行工具（仅构建时需要）

## 安装方法

```bash
# 克隆仓库
git clone [repository_url]
cd mac-statusbar-timer

# 安装到应用程序文件夹
sudo make install
```
